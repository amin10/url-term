ls <PATH>
- list all links in page indicated by PATH
- if PATH is not indicated, list all links on page

cd <PATH>
- navigate to the webopage indicated by "PATH"
- if path is relative, then should go to relative path
- if PATH is not indicated, navigate to HOME_PAGE set by export command

export <VAR_NAME>=<VAR_VALUE>
- set VAR_NAME to VAR_VALUE
